# Alexander Radar — Portfolio

A personal UX/UI design portfolio built with React, React Router, and Vite. Hosted on GitHub Pages.

🔗 **Live site:** [https://aaradar.github.io/portfolio/](https://aaradar.github.io/portfolio/)

---

## Tech Stack

- **React 18** — UI framework
- **React Router 6** — client-side routing
- **Vite** — build tool
- **GitHub Pages** — hosting

## Project Structure

```
portfolio/
├── index.html
├── vite.config.js
├── package.json
├── src/
│   ├── App.js
│   ├── App.css
│   ├── index.js
│   └── pages/
│       ├── Home.js
│       ├── Home.css
│       ├── ComingSoon.js
│       └── ComingSoon.css
└── public/
```

## Running Locally

```bash
yarn install
yarn start
```

Opens at `http://localhost:5173/portfolio/`

## Deploying

```bash
git add .
git commit -m "describe your change"
git push
yarn deploy
```
